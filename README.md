# FXSignalx — Landing Pages Demo

Static demo của 10 landing page (5 stages × 2 variants) cho [fxsignalx.com](https://fxsignalx.com).

Live preview: https://longpv.github.io/fxsignalx-landing-demo/

## Cấu trúc

```
_shared/                       # CSS dùng chung
stage-1-unaware/               # variant-a.html, variant-b.html
stage-2-problem-aware/
stage-3-solution-aware/
stage-4-product-aware/
stage-5-most-aware/
index.html                     # menu điều hướng 10 page
```

## Local preview

```bash
python3 -m http.server 8000
# mở http://localhost:8000/
```
